# ClawHub Interaction Skill

A skill for interacting with the ClawHub skill registry, including browsing, searching, and managing skills.

## Description

This skill enables interaction with the ClawHub skill registry (https://clawhub.ai). It provides functionality to browse available skills, search for specific capabilities, and manage skill installations from the registry.

## Capabilities

- Browse available skills in ClawHub
- Search for skills by functionality
- Check skill compatibility
- Access skill documentation
- Install skills from registry

## Prerequisites

- Internet connection
- OpenClaw environment
- (Optional) GitHub account for contributing skills

## Installation

This skill uses built-in OpenClaw capabilities along with web access tools.

## Usage

Use this skill when you need to:
- Find new skills to enhance functionality
- Explore available tools in the ClawHub registry
- Research skill capabilities before installation
- Understand the skill ecosystem

## Configuration

No specific configuration required.

## API

### Functions

#### `browse_skills(category)`
Browse skills by category in the ClawHub registry.

#### `search_skills(query)`
Search for skills matching a specific query or functionality.

#### `get_skill_info(skill_name)`
Retrieve detailed information about a specific skill.

#### `check_compatibility(skill_name)`
Check if a skill is compatible with the current environment.

## Examples

Usage examples:

```
Browse productivity skills in ClawHub
Search for image processing skills
Show information about the web scraping skill
Check compatibility of automation tools
```

## Notes

This skill helps users discover and leverage the broader ClawHub ecosystem to extend their OpenClaw capabilities.
