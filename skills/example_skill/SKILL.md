# Example Skill

A simple example skill to demonstrate the structure of a ClawHub skill.

## Description

This skill demonstrates the basic structure of a ClawHub skill file. It provides simple functionality to show how skills are organized and documented.

## Capabilities

- Demonstrate skill structure
- Show proper formatting
- Provide example usage

## Prerequisites

- None required

## Installation

No special installation required. This skill uses built-in OpenClaw capabilities.

## Usage

Simply call this skill to see how a typical skill operates.

## Configuration

No configuration required.
