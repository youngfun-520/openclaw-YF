# Web Search Skill

A skill that enables advanced web searching capabilities using various search engines and techniques.

## Description

This skill provides enhanced web search functionality beyond the basic tools. It allows for targeted searches, result filtering, and information extraction from web sources.

## Capabilities

- Multi-engine search (Google, Bing, DuckDuckGo)
- Result filtering and ranking
- Content summarization
- Source verification

## Prerequisites

- Internet connection
- Optional: Search engine API keys for enhanced features

## Installation

1. Ensure basic web fetching tools are available
2. Configure API keys if desired (optional)

## Usage

Use this skill when you need to:
- Perform detailed web searches
- Find specific information online
- Verify facts or claims
- Gather information from multiple sources

## Configuration

No mandatory configuration required. Optional API keys can be added for enhanced search capabilities.

## API

### Functions

#### `search(query, options)`
Perform a web search with the given query and options.

Options:
- engine: Search engine to use (google, bing, duckduckgo)
- count: Number of results to return
- freshness: Filter for recent content

#### `summarize_results(results)`
Summarize search results for easier consumption.

## Examples

Search examples:

```
Search for the latest developments in AI technology
Verify the population of Tokyo, Japan
Find articles about renewable energy trends in 2025
```

## Notes

This skill enhances the basic web search functionality with additional filtering and organization capabilities.
